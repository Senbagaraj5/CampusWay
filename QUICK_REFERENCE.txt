## New Features

| Feature | What It Does |
|---------|-------------|
| Fallback Image | Creates timestamp placeholder if camera fails |
| Fallback Location | Uses campus default location if GPS fails |
| Retry Buttons | Users can retry camera/location anytime |
| Better Errors | Specific messages instead of generic ones |
| Console Logging | Easy debugging with clear logs |
| Status Indicators | Green=ready, Amber=fallback, Spinner=loading |
